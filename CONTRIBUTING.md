# Contributing to Agent Reliability Patterns

Thanks for your interest in contributing! This project applies distributed systems reliability patterns to AI agents.

## Ways to Contribute

### 🐛 Bug Reports
- Open an issue with a clear description
- Include steps to reproduce
- Share relevant code snippets or logs

### 💡 Feature Requests
- Open an issue describing the use case
- Explain how it relates to agent reliability
- Bonus: reference relevant distributed systems patterns

### 🔧 Code Contributions

1. **Fork the repo**
2. **Create a branch**: `git checkout -b feature/your-feature`
3. **Make your changes**
4. **Run tests**: `python3 -m pytest tests/ -v`
5. **Submit a PR**

### Code Style
- Follow PEP 8
- Add docstrings to public functions
- Include type hints
- Write tests for new functionality

### Good First Issues
Look for issues labeled `good-first-issue` — these are designed to be approachable for new contributors.

## Development Setup

```bash
git clone https://github.com/hamley241/agent-reliability-patterns.git
cd agent-reliability-patterns
pip install -e ".[dev]"
python3 -m pytest tests/ -v
```
